Hosting Website easily

If you don't already have a GitHub account, go to GitHub and sign up.

  • After logging into GitHub, click the + sign in the upper right corner and select New repository.
  • Name the repository, for example, my-website. You can choose to make it public or private.
  • Initialize the repository with a README file (optional).

  1. Open your local project folder where your website files are stored (HTML, CSS, JS, images, etc.).
  2. On the GitHub repository page, click the Add file button, then select Upload files.
  3. Drag and drop your website files into the GitHub interface, or choose files to upload.
  4. Commit the changes by clicking on Commit changes.

  • Go to your GitHub repository's Settings (found in the menu at the top of your repository).
  • Scroll down to the GitHub Pages section.
  • In the Source dropdown, select the branch you want to use (usually main or master).
  • Choose the / (root) folder for the source if you're using the main repository folder.
  • After enabling GitHub Pages, a link will appear showing the URL of your live site. This will be something like https://username.github.io/repository-name.

  • Once the page refreshes, you should see a link to your website under the GitHub Pages section.
  • Click the link to visit your website live on the web.

  • If you have a custom domain, you can configure it in the GitHub Pages settings by adding a CNAME file with your domain name (e.g., www.yourdomain.com).
