Implementation Details

Pure Bootstrap

During the creation of this website, we focused on achieving a responsive and user-friendly design. Here are the key implementation steps:

  • Setting Up Bootstrap: We included the Bootstrap CSS and JS files in our project to utilize its powerful grid system and components.
  • Navbar: We used the navbar component to create a responsive navigation bar, making it easy for users to navigate our website.
  • Accordion: Implemented the accordion component to organize content into collapsible sections.
  • Alerts: Used Bootstrap alerts to provide user feedback and important messages.
  • Buttons: Applied various button styles for actions and links.
  • Card: Utilized the card component for structured presentation of content and images.
  • Carousel: Integrated a carousel to showcase images in a slideshow format.
  • Close Button: Included close button elements in alerts and modals for dismissing them.
  • Collapse: Used the collapse component to toggle visibility of elements.
  • Dropdowns: Implemented dropdowns for enhanced navigation and option selection.
  • Color & Background: Customized color and background classes for styling.
  • Utilities: Utilized various utility classes for background, borders, colors, display, flex, float, position, shadows, sizing, spacing, and text.

Using Inspect Tool

To help users better understand the HTML and CSS of our website, here's how to use the browser's Inspect tool:

  1. Right-click on any element on the webpage and select Inspect.
  2. The browser's Developer Tools will open, highlighting the HTML code of the selected element.
  3. Hover over different HTML tags in the Elements panel to see their corresponding sections highlighted on the webpage.
  4. Right-click on an HTML tag in the Elements panel and select Edit as HTML to make changes. You can copy and use the code as needed.
  5. To apply CSS styles, go to the Styles tab in the Developer Tools. You can add or modify CSS properties here to see how they affect the selected element.
  6. Experiment with different CSS properties to understand which styles and elements you can use for your own projects.
